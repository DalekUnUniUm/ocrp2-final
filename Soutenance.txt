Expliquez mécaniques GitHub -> Faire au moins une branche 



Découpez le code en 4 versions

	- Analyse du code
	- Passer le code en POO 
	- Ajout de nouvelles classe
	- Code final 


git clone = Cloner fichier de github

git add . = Ajout de tout les fichiers 

git status = Status du git 

git commit = mettre dans le repo local (-m pour ajout de commentaire) 

git push = mettre dans le repo github

git branch = liste toute les branche du projet
